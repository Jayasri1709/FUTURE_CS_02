# Phishing Email Detection & Awareness System

## Overview
This project focuses on identifying phishing emails by analyzing sender information, email content, suspicious links, and phishing indicators.

## Objective
To classify emails as Safe, Suspicious, or Phishing and provide awareness guidelines to help users avoid phishing attacks.

## Tools Used
- Email Samples
- Browser Analysis
- Microsoft Word
- PDF Documentation

## Key Findings
- Fake sender domains are a common phishing technique.
- Urgent messages are often used to create panic.
- Suspicious links should always be verified before clicking.

## Prevention Guidelines
- Verify sender email addresses.
- Avoid clicking unknown links.
- Enable Multi-Factor Authentication.
- Report suspicious emails.

## Outcome
Successfully analyzed sample emails and identified phishing indicators while providing security awareness recommendations.
